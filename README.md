# 👕 FitSize

온라인 쇼핑할 때 옷 사이즈, 입어보고 결정하세요.

신체 정보와 옷 실측치를 입력하면 2D 아바타 위에 옷을 입혀서 핏을 시각화해줍니다.

## 주요 기능

### 🧍 체형 입력 (2가지 방식)
- **직접 입력** — 키, 몸무게, 어깨너비 등 신체 치수를 아는 경우
- **내 옷으로 추정** — 이미 갖고 있는 옷의 실측치 + 착용감(딱맞음/여유/빡빡)을 입력하면 체형을 역추정

### 👔 옷 피팅 시뮬레이션
- 티셔츠, 긴팔, 자켓, 바지, 원피스 카테고리 지원
- 옷 실측치(총장, 어깨, 가슴둘레, 소매 등) 입력
- 아바타에 옷을 입혀 부위별 피팅 상태 시각화
  - 🟢 적당 (-1cm ~ +3cm)
  - 🟡 여유 (+3cm 이상)
  - 🔴 빡빡 (-1cm 이하)

### 📦 옷장 관리
- 입력한 옷을 이름 붙여 저장
- 저장된 옷들로 체형 프로필 자동 갱신
- 재방문 시 저장된 프로필로 바로 피팅 시작

### 📤 데이터 이동
- JSON 파일로 내보내기/불러오기 지원
- 다른 기기에서도 데이터 이전 가능
- 현재 localStorage 기반, 추후 클라우드 동기화 예정

### 🛒 상품 추천
- 피팅 결과에 맞는 사이즈 가이드 제공

## 기술 스택

- React 19 + TypeScript
- Vite
- Tailwind CSS 4
- HTML Canvas 2D

## 실행

```bash
npm install
npm run dev
```
